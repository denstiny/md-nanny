# md-nanny
> Strengthen the markdown in neovim
## install
### Using a plugin manager

Using plug:
```lua
Plug 'nvim-zh/md-nanny'
```

Using Packer:
```lua
return require("packer").startup( function(use)
 	use "nvim-zh/md-nanny"
 end
)
``` 

## Setup

```lua
require('md-nanny')
```
